ค30107 · The Derivative Function · D4

Function Analysis I การวิเคราะห์ฟังก์ชัน I

Use f ′ and f ″ to find intervals of increase/decrease, local extrema, and concavity. ใช้ f ′ และ f ″ เพื่อหาช่วงที่เพิ่ม/ลด, จุดสุดขีดเฉพาะที่ และความเว้า

Part 1 — First Derivative: Increasing & Decreasing

Key facts f ′(x) > 0 on an interval  →  f is increasing there
f ′(x) < 0 on an interval  →  f is decreasing there
f ′(c) = 0 or f ′(c) undefined  →  critical point at x = c

First Derivative Test for Local Extrema

At a critical point c, look at the sign change of f ′ as x passes through c:

Sign of f ′ left of cSign of f ′ right of cConclusion
+−Local maximum at c
−+Local minimum at c
+ or − (no change)sameNeither (saddle/inflection)
The sign chart method: Find all critical points. Pick test values in each interval between them. Evaluate the sign of f ′ (positive or negative). Read off where f rises or falls.

ส่วนที่ 1 — อนุพันธ์อันดับหนึ่ง: เพิ่มและลด

ข้อเท็จจริงหลัก f ′(x) > 0 บนช่วง  →  f เพิ่มในช่วงนั้น
f ′(x) < 0 บนช่วง  →  f ลดในช่วงนั้น
f ′(c) = 0 หรือ f ′(c) ไม่มีนิยาม  →  จุดวิกฤต ที่ x = c

การทดสอบด้วยอนุพันธ์อันดับหนึ่งสำหรับค่าสุดขีดเฉพาะที่

ที่จุดวิกฤต c ดูการเปลี่ยนเครื่องหมายของ f ′ เมื่อ x ผ่าน c:

เครื่องหมาย f ′ ซ้าย cเครื่องหมาย f ′ ขวา cสรุป
+−ค่าสูงสุดเฉพาะที่ที่ c
−+ค่าต่ำสุดเฉพาะที่ที่ c
+ หรือ − (ไม่เปลี่ยน)เดิมไม่ใช่ค่าสุดขีด
วิธีตาราง/แผนภาพเครื่องหมาย: หาจุดวิกฤตทั้งหมด เลือกค่าทดสอบในแต่ละช่วงระหว่างจุดวิกฤต หาเครื่องหมายของ f ′ (บวกหรือลบ) แล้วอ่านว่า f เพิ่มหรือลดที่ไหน

Part 2 — Second Derivative: Concavity

Key facts f ″(x) > 0 on an interval  →  f is concave up (∪ shaped)
f ″(x) < 0 on an interval  →  f is concave down (∩ shaped)
f ″ changes sign at x = c  →  inflection point at c

Second Derivative Test for Local Extrema

If f ′(c) = 0:

f ″(c)Conclusion
f ″(c) > 0Local minimum (bowl opens up)
f ″(c) < 0Local maximum (bowl opens down)
f ″(c) = 0Inconclusive — use first derivative test

ส่วนที่ 2 — อนุพันธ์อันดับสอง: ความเว้า

ข้อเท็จจริงหลัก f ″(x) > 0 บนช่วง  →  f เว้าขึ้น (รูปทรง ∪)
f ″(x) < 0 บนช่วง  →  f เว้าลง (รูปทรง ∩)
f ″ เปลี่ยนเครื่องหมายที่ x = c  →  จุดเปลี่ยนความเว้าที่ c

การทดสอบด้วยอนุพันธ์อันดับสองสำหรับค่าสุดขีดเฉพาะที่

ถ้า f ′(c) = 0:

f ″(c)สรุป
f ″(c) > 0ค่าต่ำสุดเฉพาะที่ (ชามหงาย)
f ″(c) < 0ค่าสูงสุดเฉพาะที่ (ชามควํ่า)
f ″(c) = 0สรุปไม่ได้ — ใช้การทดสอบด้วยอนุพันธ์อันดับหนึ่ง

Worked Example

Fully analyse \(f(x) = x^3 - 3x^2 - 9x + 5\).

Step 1 — Find critical points

\(f'(x) = 3x^2 - 6x - 9 = 3(x^2 - 2x - 3) = 3(x-3)(x+1)\)

Critical points: \(x = 3\) and \(x = -1\).

Step 2 — Sign chart for f ′

IntervalTest valuef ′ signf behavior
\((-\infty, -1)\)\(x=-2\)\(3(-5)(-1)=+\)Increasing ↑
\((-1, 3)\)\(x=0\)\(3(-3)(1)=-\)Decreasing ↓
\((3, \infty)\)\(x=4\)\(3(1)(5)=+\)Increasing ↑

Conclusion: Local max at \(x = -1\), local min at \(x = 3\).

Step 3 — Concavity via f ″

\(f''(x) = 6x - 6 = 6(x-1)\)

\(f''(x) = 0\) at \(x = 1\).

  • \(x < 1\): \(f'' < 0\) → concave down
  • \(x > 1\): \(f'' > 0\) → concave up

Inflection point at \(x = 1\) (f″ changes sign).

ตัวอย่างที่ผ่านการคำนวณ

วิเคราะห์ \(f(x) = x^3 - 3x^2 - 9x + 5\) อย่างสมบูรณ์

ขั้น 1 — หาจุดวิกฤต

\(f'(x) = 3x^2 - 6x - 9 = 3(x^2 - 2x - 3) = 3(x-3)(x+1)\)

จุดวิกฤต: \(x = 3\) และ \(x = -1\)

ขั้น 2 — ตารางเครื่องหมายของ f ′

ช่วงค่าทดสอบเครื่องหมาย f ′พฤติกรรม f
\((-\infty, -1)\)\(x=-2\)\(3(-5)(-1)=+\)เพิ่ม ↑
\((-1, 3)\)\(x=0\)\(3(-3)(1)=-\)ลด ↓
\((3, \infty)\)\(x=4\)\(3(1)(5)=+\)เพิ่ม ↑

สรุป: ค่าสูงสุดเฉพาะที่ที่ \(x = -1\), ค่าต่ำสุดเฉพาะที่ที่ \(x = 3\)

ขั้น 3 — ความเว้าผ่าน f ″

\(f''(x) = 6x - 6 = 6(x-1)\)

\(f''(x) = 0\) ที่ \(x = 1\)

  • \(x < 1\): \(f'' < 0\) → เว้าลง
  • \(x > 1\): \(f'' > 0\) → เว้าขึ้น

จุดเปลี่ยนความเว้าที่ \(x = 1\) (f″ เปลี่ยนเครื่องหมาย)
