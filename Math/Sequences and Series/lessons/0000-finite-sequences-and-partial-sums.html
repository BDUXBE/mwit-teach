Finite Sequences, Sigma Notation & Partial Sums ลำดับจำกัด สัญลักษณ์ซิกมา และผลรวมย่อย

Goal: Lock in the finite-sequence formulas before moving to infinite ones. The geometric partial sum formula Sn = a(1−rⁿ)/(1−r) reappears directly in the geometric series lesson. Know it cold. จุดมุ่งหมาย: ทบทวนสูตรลำดับจำกัดให้แม่นก่อนเรียนอนุกรมอนันต์ สูตรผลรวมย่อยเรขาคณิต Sn = a(1−rⁿ)/(1−r) จะปรากฏโดยตรงในบทอนุกรมเรขาคณิต ต้องจำให้ขึ้นใจ

1. Sequences — Quick Definition

A sequence is a function whose domain is the positive integers. We write the terms as a1, a2, a3, …, an, …

Two ways to define a sequence Explicit (แบบชัดแจ้ง): an = 2n − 1 gives 1, 3, 5, 7, 9, … Recursive (แบบวนซ้ำ): F0 = 1, F1 = 1, Fk = Fk−2 + Fk−1 (Fibonacci)

1. ลำดับ — นิยามเบื้องต้น

ลำดับ คือฟังก์ชันที่มีโดเมนเป็นจำนวนเต็มบวก เราเขียนพจน์เป็น a1, a2, a3, …, an, …

สองวิธีในการนิยามลำดับ Explicit (แบบชัดแจ้ง): an = 2n − 1 gives 1, 3, 5, 7, 9, … Recursive (แบบวนซ้ำ): F0 = 1, F1 = 1, Fk = Fk−2 + Fk−1 (Fibonacci)

2. Sigma Notation (Summation Notation)

Definition Σk=1n ak = a1 + a2 + a3 + ··· + an k = index of summation 1 = lower limit, n = upper limit

Key summation formulas (memorize these)

Standard formulas Σi=1n c = cn (constant sum) Σi=1n i = n(n+1)/2 (sum of first n integers) Σi=1n i² = n(n+1)(2n+1)/6 Σi=1n i³ = [n(n+1)/2]² = (Σ i)²
Why these matter: they let you collapse long sums into closed-form expressions, which is exactly what you need when computing partial sums of sequences.

2. สัญลักษณ์ซิกมา (สัญลักษณ์การบวก)

นิยาม Σk=1n ak = a1 + a2 + a3 + ··· + an k = ดัชนีการบวก 1 = ขอบล่าง, n = ขอบบน

สูตรการบวกที่ต้องจำ (ท่องให้ขึ้นใจ)

สูตรมาตรฐาน Σi=1n c = cn (ผลรวมของค่าคงที่) Σi=1n i = n(n+1)/2 (ผลรวม n จำนวนเต็มแรก) Σi=1n i² = n(n+1)(2n+1)/6 Σi=1n i³ = [n(n+1)/2]² = (Σ i)²
เหตุใดสูตรเหล่านี้จึงสำคัญ: ช่วยให้เขียนผลรวมที่ยาวให้อยู่ในรูปปิดได้ ซึ่งจำเป็นมากเมื่อต้องคำนวณผลรวมย่อยของลำดับ

3. Arithmetic Sequences

Arithmetic Sequence a, a+d, a+2d, a+3d, … nth term: an = a + (n−1)d a = first term, d = common difference

Partial Sum of an Arithmetic Sequence

Two equivalent formulas Sn = n/2 · (2a + (n−1)d) Sn = n · (a + an)/2 ← "average of first and last × n"

Example

Sum 3 + 7 + 11 + ··· + 159 a = 3, d = 4. Find n: an = 3 + (n−1)·4 = 159 → n = 40 S40 = 40 · (3 + 159)/2 = 40 · 81 = 3,240

3. ลำดับเลขคณิต

ลำดับเลขคณิต a, a+d, a+2d, a+3d, … พจน์ที่ n: an = a + (n−1)d a = พจน์แรก, d = ผลต่างร่วม

ผลรวมย่อยของลำดับเลขคณิต

สูตรสองแบบที่เทียบเท่ากัน Sn = n/2 · (2a + (n−1)d) Sn = n · (a + an)/2 ← "ค่าเฉลี่ยของพจน์แรกและพจน์สุดท้าย × n"

ตัวอย่าง

หา 3 + 7 + 11 + ··· + 159 a = 3, d = 4 หา n: an = 3 + (n−1)·4 = 159 → n = 40 S40 = 40 · (3 + 159)/2 = 40 · 81 = 3,240

4. Geometric Sequences

Geometric Sequence a, ar, ar², ar³, … nth term: an = arn−1 a = first term, r = common ratio (r ≠ 0)

Partial Sum of a Geometric Sequence

Finite geometric sum Sn = a(1 − rn) / (1 − r) (r ≠ 1) Derivation: write Sn, multiply by r, subtract → Sn − r·Sn = a − arn Sn(1 − r) = a(1 − rn)
This formula reappears in S3! The infinite geometric series sum S = a/(1−r) is exactly this formula with rn → 0 as n → ∞ (when |r| < 1). If you know this derivation, you understand why the infinite sum formula works.

Example

Sum 1 + 0.7 + 0.49 + 0.343 + 0.2401 (first five terms) a = 1, r = 0.7, n = 5 S5 = 1·(1 − 0.7⁵)/(1 − 0.7) = (1 − 0.16807)/0.3 = 0.83193/0.3 ≈ 2.773

4. ลำดับเรขาคณิต

ลำดับเรขาคณิต a, ar, ar², ar³, … พจน์ที่ n: an = arn−1 a = พจน์แรก, r = อัตราส่วนร่วม (r ≠ 0)

ผลรวมย่อยของลำดับเรขาคณิต

ผลรวมย่อยเรขาคณิตจำกัด Sn = a(1 − rn) / (1 − r) (r ≠ 1) วิธีพิสูจน์: เขียน Sn คูณด้วย r แล้วลบ → Sn − r·Sn = a − arn Sn(1 − r) = a(1 − rn)
สูตรนี้จะปรากฏอีกครั้งในบท S3! ผลรวมอนุกรมเรขาคณิตอนันต์ S = a/(1−r) คือสูตรนี้นั่นเองเมื่อ rn → 0 ขณะ n → ∞ (เมื่อ |r| < 1) ถ้าเข้าใจวิธีพิสูจน์นี้ จะเข้าใจว่า ทำไม สูตรผลรวมอนันต์ถึงทำงานได้

ตัวอย่าง

หา 1 + 0.7 + 0.49 + 0.343 + 0.2401 (5 พจน์แรก) a = 1, r = 0.7, n = 5 S5 = 1·(1 − 0.7⁵)/(1 − 0.7) = (1 − 0.16807)/0.3 = 0.83193/0.3 ≈ 2.773
