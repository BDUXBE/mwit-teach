Applications of Geometric Series — Interest & Annuities การประยุกต์อนุกรมเรขาคณิต — ดอกเบี้ยและเงินรายงวด

After this lesson: you can model compound interest and fixed annuity problems using geometric sequences and series — the PISA-style application problems on the quiz. หลังจากเรียนบทนี้: นักเรียนสามารถสร้างแบบจำลองปัญหาดอกเบี้ยทบต้นและเงินรายงวดคงที่โดยใช้ลำดับและอนุกรมเรขาคณิต — โจทย์ประยุกต์แบบ PISA ในแบบทดสอบ

1. Compound Interest — Geometric Sequence

When interest is compounded, each period multiplies the balance by a fixed ratio. This is a geometric sequence.

Compound Interest Formula An = P · (1 + r)n P = principal (initial deposit) r = interest rate per period (as a decimal) n = number of periods An = balance after n periods

Example — Single Deposit

Example A Deposit ฿10,000 at 5% annual interest, compounded yearly. What is the balance after 10 years? A10 = 10,000 · (1.05)10 ≈ 10,000 · 1.6289 ≈ ฿16,289
Why geometric? Each year's balance = previous balance × 1.05. That constant multiplier (1 + r) is the common ratio r of the geometric sequence.

1. ดอกเบี้ยทบต้น — ลำดับเรขาคณิต

เมื่อดอกเบี้ยทบต้น แต่ละงวดจะคูณยอดเงินด้วยอัตราส่วนคงที่ นี่คือลำดับเรขาคณิต

สูตรดอกเบี้ยทบต้น An = P · (1 + r)n P = เงินต้น (เงินฝากเริ่มต้น) r = อัตราดอกเบี้ยต่องวด (ในรูปทศนิยม) n = จำนวนงวด An = ยอดเงินหลังจาก n งวด

ตัวอย่าง — ฝากครั้งเดียว

ตัวอย่าง ก ฝากเงิน ฿10,000 ที่ดอกเบี้ย 5% ต่อปี คิดดอกเบี้ยแบบปีต่อปี ยอดเงินหลัง 10 ปีเท่าไร? A10 = 10,000 · (1.05)10 ≈ 10,000 · 1.6289 ≈ ฿16,289
ทำไมถึงเป็นเรขาคณิต? ยอดเงินแต่ละปี = ยอดเงินปีก่อน × 1.05 ตัวคูณคงที่ (1 + r) นี้คืออัตราส่วนร่วมของลำดับเรขาคณิต

2. Repeated Deposits — Geometric Series

If you deposit the same amount P at the end of every period, the total accumulated value is a finite geometric series — each deposit earns interest for a different number of periods.

Future Value of an Ordinary Annuity FV = P · [(1+r)n − 1] / r P = payment per period r = interest rate per period n = total number of payments FV = total value at the END of period n

Derivation — where does the formula come from?

Step by step Deposit P at end of period 1 → grows to P(1+r)n−1 by period n Deposit P at end of period 2 → grows to P(1+r)n−2 by period n ··· Deposit P at end of period n → stays as P (no time to grow) FV = P(1+r)n−1 + P(1+r)n−2 + ··· + P(1+r) + P = P · [(1+r)n − 1] / r (geometric series sum, first term = P, ratio = 1+r)

Example — Monthly Savings

Example B Deposit ฿2,000 at the end of every month for 3 years (36 months). Annual interest = 6%, so monthly rate r = 0.06/12 = 0.005. FV = 2,000 · [(1.005)36 − 1] / 0.005 = 2,000 · [1.1967 − 1] / 0.005 = 2,000 · 0.1967 / 0.005 = 2,000 · 39.34 ≈ ฿78,680

2. การฝากซ้ำ — อนุกรมเรขาคณิต

ถ้าฝากเงิน P เท่ากันที่ ปลายของทุกงวด มูลค่าสะสมรวมคืออนุกรมเรขาคณิตจำกัด — แต่ละเงินฝากได้รับดอกเบี้ยเป็นจำนวนงวดที่ต่างกัน

มูลค่าอนาคตของเงินรายงวดสามัญ FV = P · [(1+r)n − 1] / r P = เงินชำระต่องวด r = อัตราดอกเบี้ยต่องวด n = จำนวนการชำระทั้งหมด FV = มูลค่ารวมที่ปลายงวดที่ n

การพิสูจน์ — สูตรนี้มาจากไหน?

ทีละขั้นตอน ฝาก P ที่ปลายงวด 1 → เติบโตเป็น P(1+r)n−1 ถึงงวดที่ n ฝาก P ที่ปลายงวด 2 → เติบโตเป็น P(1+r)n−2 ถึงงวดที่ n ··· ฝาก P ที่ปลายงวด n → คงเป็น P (ไม่มีเวลาเติบโต) FV = P(1+r)n−1 + P(1+r)n−2 + ··· + P(1+r) + P = P · [(1+r)n − 1] / r (ผลรวมอนุกรมเรขาคณิต พจน์แรก = P อัตราส่วน = 1+r)

ตัวอย่าง — การออมรายเดือน

ตัวอย่าง ข ฝาก ฿2,000 ที่ปลายทุกเดือน เป็นเวลา 3 ปี (36 เดือน) ดอกเบี้ยต่อปี = 6% ดังนั้นอัตราดอกเบี้ยรายเดือน r = 0.06/12 = 0.005 FV = 2,000 · [(1.005)36 − 1] / 0.005 = 2,000 · [1.1967 − 1] / 0.005 = 2,000 · 0.1967 / 0.005 = 2,000 · 39.34 ≈ ฿78,680

3. Present Value of an Annuity

"What lump sum today is equivalent to receiving P per period for n periods?" Each future payment P must be discounted back to today.

Present Value of an Ordinary Annuity PV = P · [1 − (1+r)−n] / r

Example — Loan Repayment

Example C You want to borrow money and repay ฿5,000/month for 24 months. Monthly rate r = 0.005 (6% annual). PV = 5,000 · [1 − (1.005)−24] / 0.005 = 5,000 · [1 − 0.8872] / 0.005 = 5,000 · 0.1128 / 0.005 = 5,000 · 22.56 ≈ ฿112,800 ← maximum loan amount

3. มูลค่าปัจจุบันของเงินรายงวด

"เงินก้อนเดียวในวันนี้เท่าไรที่เทียบเท่ากับการรับ P ต่องวด เป็นเวลา n งวด?" เงินชำระในอนาคตแต่ละงวด P ต้องคิดลดกลับมาเป็นมูลค่าปัจจุบัน

มูลค่าปัจจุบันของเงินรายงวดสามัญ PV = P · [1 − (1+r)−n] / r

ตัวอย่าง — การผ่อนชำระเงินกู้

ตัวอย่าง ค ต้องการกู้เงินและผ่อนชำระ ฿5,000/เดือน เป็นเวลา 24 เดือน อัตราดอกเบี้ยรายเดือน r = 0.005 (6% ต่อปี) PV = 5,000 · [1 − (1.005)−24] / 0.005 = 5,000 · [1 − 0.8872] / 0.005 = 5,000 · 0.1128 / 0.005 = 5,000 · 22.56 ≈ ฿112,800 ← วงเงินกู้สูงสุด
