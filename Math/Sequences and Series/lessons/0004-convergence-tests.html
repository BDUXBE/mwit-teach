Convergence Tests การทดสอบการลู่เข้า

After this lesson: given any series from the course, you can choose the right test and correctly classify it as convergent or divergent. หลังจากเรียนบทนี้: เมื่อกำหนดอนุกรมใดๆ จากบทเรียนนี้ นักเรียนสามารถเลือกการทดสอบที่เหมาะสมและจำแนกได้อย่างถูกต้องว่าลู่เข้าหรือแยกออก

1. The Decision Flowchart

Always work top-to-bottom. Stop as soon as a test gives a definitive answer.

Which test to use — in order Step 1: Divergence Test If lim a_n ≠ 0 → DIVERGES. Done. If lim a_n = 0 → inconclusive, keep going. Step 2: Is it geometric? (a·r^n form) If |r| < 1 → CONVERGES, sum = a/(1−r). If |r| ≥ 1 → DIVERGES. Step 3: Does it telescope? (partial fractions give f(n) − f(n+1)) Write out S_n, find the pattern, take the limit. Step 4: Does it look like Σ 1/n^p? p > 1 → CONVERGES (p-series) p ≤ 1 → DIVERGES (includes harmonic: p = 1)
For ค30107: these four steps cover virtually every series you will see on the quiz. Master them in order.

1. แผนผังการตัดสินใจ

ทำงานจากบนลงล่างเสมอ หยุดทันทีที่การทดสอบให้คำตอบที่ชัดเจน

จะใช้การทดสอบใด — เรียงตามลำดับ ขั้น 1: การทดสอบการแยกออก ถ้า lim a_n ≠ 0 → แยกออก จบ ถ้า lim a_n = 0 → สรุปไม่ได้ ดำเนินต่อ ขั้น 2: เป็นเรขาคณิตหรือไม่? (รูป a·r^n) ถ้า |r| < 1 → ลู่เข้า ผลรวม = a/(1−r) ถ้า |r| ≥ 1 → แยกออก ขั้น 3: เป็นเทเลสโคปหรือไม่? (แยกเศษย่อยได้ f(n) − f(n+1)) เขียน S_n ออกมา หารูปแบบ แล้วหาลิมิต ขั้น 4: มีรูปแบบ Σ 1/n^p หรือไม่? p > 1 → ลู่เข้า (p-series) p ≤ 1 → แยกออก (รวมถึงฮาร์โมนิก: p = 1)
สำหรับ ค30107: สี่ขั้นตอนนี้ครอบคลุมอนุกรมแทบทุกชนิดที่จะเจอในแบบทดสอบ เรียนรู้ให้แม่นตามลำดับ

2. Divergence Test (nth-Term Test)

Rule If limn→∞ an ≠ 0 → Σ an diverges. WARNING: If lim a_n = 0, this test is INCONCLUSIVE. (Example: 1/n → 0 but Σ 1/n still diverges.)

Examples

Example — Diverges immediately Σ n/(2n+1) lim a_n = lim n/(2n+1) = 1/2 ≠ 0 → Diverges. (No further work needed.)

2. การทดสอบการแยกออก (การทดสอบพจน์ที่ n)

กฎ If limn→∞ an ≠ 0 → Σ an diverges. คำเตือน: ถ้า lim a_n = 0 การทดสอบนี้ให้ผลไม่ชี้ขาด (ตัวอย่าง: 1/n → 0 แต่ Σ 1/n ยังแยกออก)

ตัวอย่าง

ตัวอย่าง — แยกออกทันที Σ n/(2n+1) lim a_n = lim n/(2n+1) = 1/2 ≠ 0 → แยกออก (ไม่ต้องทำงานเพิ่มเติม)

3. Geometric Series Test

Rule Σ a·rn converges iff |r| < 1, sum = a/(1−r). Otherwise diverges.
Identifying the ratio r r = an+1 / an (check it is constant — same for every n)

3. การทดสอบอนุกรมเรขาคณิต

กฎ Σ a·rn ลู่เข้าก็ต่อเมื่อ |r| < 1 ผลรวม = a/(1−r) มิฉะนั้นแยกออก
การหาอัตราส่วน r r = an+1 / an (ตรวจสอบว่าคงที่ — เหมือนกันทุก n)

4. Telescoping Series

When to suspect telescoping Numerator is constant, denominator is a product of two consecutive linear terms. Examples: 1/(n(n+1)), 1/((n+1)(n+3)), 1/(n²−1) Method: 1. Partial fractions to write a_n = f(n) − f(n+k) 2. Write out S_n explicitly 3. Identify what survives after cancellation 4. Take lim S_n
Example — Σ 1/((2n−1)(2n+1)) Partial fractions: 1/((2n−1)(2n+1)) = (1/2)[1/(2n−1) − 1/(2n+1)] S_n = (1/2)[(1/1 − 1/3) + (1/3 − 1/5) + ··· + (1/(2n−1) − 1/(2n+1))] = (1/2)[1 − 1/(2n+1)] lim S_n = (1/2)(1) = 1/2

4. อนุกรมเทเลสโคป

เมื่อใดควรสังเกตว่าเป็นเทเลสโคป ตัวเศษเป็นค่าคงที่ ตัวส่วนเป็นผลคูณของพจน์เชิงเส้นสองพจน์ที่ต่อเนื่องกัน Examples: 1/(n(n+1)), 1/((n+1)(n+3)), 1/(n²−1) วิธีการ: 1. แยกเศษย่อยเพื่อเขียน a_n = f(n) − f(n+k) 2. เขียน S_n ออกมาอย่างชัดเจน 3. ระบุสิ่งที่เหลืออยู่หลังการหักล้าง 4. หา lim S_n
ตัวอย่าง — Σ 1/((2n−1)(2n+1)) แยกเศษย่อย: 1/((2n−1)(2n+1)) = (1/2)[1/(2n−1) − 1/(2n+1)] S_n = (1/2)[(1/1 − 1/3) + (1/3 − 1/5) + ··· + (1/(2n−1) − 1/(2n+1))] = (1/2)[1 − 1/(2n+1)] lim S_n = (1/2)(1) = 1/2

5. p-Series

p-Series Rule Σn=1∞ 1/np converges if p > 1 diverges if p ≤ 1 Common cases: p = 1: Σ 1/n (harmonic) → DIVERGES p = 2: Σ 1/n² → CONVERGES (= π²/6) p = 1/2: Σ 1/√n → DIVERGES

5. p-อนุกรม

กฎ p-อนุกรม Σn=1∞ 1/np ลู่เข้า ถ้า p > 1 แยกออก ถ้า p ≤ 1 กรณีที่พบบ่อย: p = 1: Σ 1/n (ฮาร์โมนิก) → แยกออก p = 2: Σ 1/n² → ลู่เข้า (= π²/6) p = 1/2: Σ 1/√n → แยกออก

6. Side-by-Side Comparison

Series Test Used Result
Σ n/(2n+1) Divergence Test (lim = 1/2) Diverges
Σ (2/3)n Geometric (r = 2/3) Converges = 2
Σ 1/(n(n+1)) Telescoping Converges = 1
Σ 1/n p-series (p = 1) Diverges
Σ 1/n³ p-series (p = 3) Converges
Σ 1/√n p-series (p = 1/2) Diverges

6. ตารางเปรียบเทียบ

อนุกรม การทดสอบที่ใช้ ผลลัพธ์
Σ n/(2n+1) การทดสอบการแยกออก (lim = 1/2) แยกออก
Σ (2/3)n เรขาคณิต (r = 2/3) ลู่เข้า = 2
Σ 1/(n(n+1)) เทเลสโคป ลู่เข้า = 1
Σ 1/n p-อนุกรม (p = 1) แยกออก
Σ 1/n³ p-อนุกรม (p = 3) ลู่เข้า
Σ 1/√n p-อนุกรม (p = 1/2) แยกออก
