Infinite Sequences — Limits & Convergence ลำดับอนันต์ — ลิมิตและการลู่เข้า

After this lesson: you can determine whether a given sequence converges or diverges, and find the limit when it does. This maps to Learning Objective 1 — worth 10 points on the quiz. หลังจากเรียนบทนี้: นักเรียนสามารถตัดสินได้ว่าลำดับที่กำหนดมีลิมิต (ลู่เข้า) หรือไม่มีลิมิต (แยกออก) และหาค่าลิมิตเมื่อมีอยู่ ตรงกับ วัตถุประสงค์การเรียนรู้ข้อ 1 — คะแนน 10 คะแนนในแบบทดสอบ

1. What is an Infinite Sequence?

A finite sequence stops: 2, 4, 6, 8, 10. An infinite sequence goes on forever, indexed by every positive integer:

Definition { an } = a1, a2, a3, … , an, …

Think of it as a function f(n) defined on the positive integers. For example, an = 1/n gives the sequence 1, 1/2, 1/3, 1/4, …

Key question for every sequence: Does an settle toward a fixed value as n grows without bound — or does it wander, oscillate, or blow up?

1. ลำดับอนันต์คืออะไร?

ลำดับจำกัด มีจุดสิ้นสุด: 2, 4, 6, 8, 10 แต่ ลำดับอนันต์ ดำเนินไปตลอด โดยมีดัชนีเป็นจำนวนเต็มบวกทุกตัว:

นิยาม { an } = a1, a2, a3, … , an, …

คิดว่ามันคือฟังก์ชัน f(n) ที่นิยามบนจำนวนเต็มบวก ตัวอย่างเช่น an = 1/n ให้ลำดับ 1, 1/2, 1/3, 1/4, …

คำถามหลักสำหรับทุกลำดับ: an เข้าใกล้ค่าคงที่เมื่อ n โตไปไม่สิ้นสุด หรือมันเดินเตร่ แกว่งกวัด หรือระเบิดออกไปแทน?

2. The Limit of a Sequence

The limit of a sequence is the value that an approaches as n → ∞.

Notation limn→∞ an = L

This means: for any tiny margin ε > 0 you choose, there is an index N beyond which every term satisfies |an − L| < ε. In plain English: the terms get arbitrarily close to L and stay there.

Convergent vs. Divergent

TermMeaningExample
Converges lim an = L exists and is finite an = 1/n → 0
Diverges to ±∞ Terms grow without bound an = n² → +∞
Diverges (oscillates) Terms keep changing, no single limit an = (−1)n → no limit
Common mistake: (−1)n does not converge to 0. It alternates between +1 and −1 forever — the limit does not exist. You must be able to spot oscillating sequences immediately.

2. ลิมิตของลำดับ

ลิมิตของลำดับคือค่าที่ an เข้าใกล้เมื่อ n → ∞

สัญลักษณ์ limn→∞ an = L

หมายความว่า: สำหรับค่าประมาณ ε > 0 ใดๆ ที่เลือก จะมีดัชนี N ที่ทุกพจน์หลังจากนั้นสอดคล้อง |an − L| < ε กล่าวง่ายๆ คือ: พจน์ต่างๆ เข้าใกล้ L มากขึ้นเรื่อยๆ และอยู่ใกล้ L ต่อไป

ลำดับลู่เข้า กับ ลำดับแยกออก

คำศัพท์ความหมายตัวอย่าง
ลู่เข้า lim an = L มีอยู่และเป็นจำนวนจำกัด an = 1/n → 0
แยกออกสู่ ±∞ พจน์โตไปไม่มีขอบเขต an = n² → +∞
แยกออก (แกว่งกวัด) พจน์เปลี่ยนแปลงตลอด ไม่มีลิมิตเดียว an = (−1)n → no limit
ข้อผิดพลาดที่พบบ่อย: (−1)n ไม่ลู่เข้าสู่ 0 มันสลับระหว่าง +1 และ −1 ตลอดไป — ลิมิตไม่มีอยู่ ต้องสังเกตลำดับแกว่งกวัดได้ทันที

3. Key Theorems (Thm 1.2.4 · 1.2.8 · 1.2.12 · 1.2.16)

Thm 1.2.4 — Continuous limit carries over to sequences If limx→∞ f(x) = L and an = f(n) for integer n, then limn→∞ an = L. Why it matters: lets you apply L'Hôpital's Rule to sequences. If lim f(x)/g(x) needs L'Hôpital, the sequence version gets the same answer.
Thm 1.2.8 — Limit Laws (memorise these six) If lim an = A and lim bn = B: Scalar: lim (c · an) = c · A Sum: lim (an ± bn) = A ± B Product: lim (an · bn) = A · B Quotient: lim (an / bn) = A / B (B ≠ 0) Power: lim (an)p = Ap (A > 0) Constant: lim c = c
Thm 1.2.12 — Even/Odd subsequence criterion { an } converges to L ⟺ both lim a2n = L AND lim a2n−1 = L. Use this to prove (−1)n diverges: even terms → +1, odd terms → −1, limits differ → no overall limit.
Thm 1.2.16 — Absolute-value squeeze If limn→∞ |an| = 0, then limn→∞ an = 0. Example: an = (−1)n / 2n |an| = 1/2n → 0, so an → 0. ✓ (The alternating sign doesn't matter once |an| → 0.)
Exam shortcut: For any sequence with a factor (−1)n or (−1)n+1, ignore the sign and evaluate |an|. If |an| → 0, the sequence converges to 0. If |an| → anything else, you need Thm 1.2.12 (even/odd check).

3. ทฤษฎีบทสำคัญ (ทฤษฎีบท 1.2.4 · 1.2.8 · 1.2.12 · 1.2.16)

ทฤษฎีบท 1.2.4 — ลิมิตต่อเนื่องถ่ายทอดสู่ลำดับ If limx→∞ f(x) = L and an = f(n) for integer n, then limn→∞ an = L. เหตุใดจึงสำคัญ: ช่วยให้ใช้กฎของโลปีตาลกับลำดับได้ ถ้า lim f(x)/g(x) ต้องใช้โลปีตาล ลำดับก็ได้ผลเฉลยเดียวกัน
ทฤษฎีบท 1.2.8 — กฎของลิมิต (จำหกข้อนี้) If lim an = A and lim bn = B: คูณสเกลาร์: lim (c · an) = c · A บวก/ลบ: lim (an ± bn) = A ± B คูณ: lim (an · bn) = A · B หาร: lim (an / bn) = A / B (B ≠ 0) ยกกำลัง: lim (an)p = Ap (A > 0) ค่าคงที่: lim c = c
ทฤษฎีบท 1.2.12 — เกณฑ์ลำดับย่อยคู่/คี่ { an } converges to L ⟺ both lim a2n = L AND lim a2n−1 = L. ใช้พิสูจน์ว่า (−1)n แยกออก: พจน์คู่ → +1 พจน์คี่ → −1 ลิมิตต่างกัน → ไม่มีลิมิตรวม
ทฤษฎีบท 1.2.16 — การบีบด้วยค่าสัมบูรณ์ If limn→∞ |an| = 0, then limn→∞ an = 0. Example: an = (−1)n / 2n |an| = 1/2n → 0, so an → 0. ✓ (เครื่องหมายสลับไม่สำคัญเมื่อ |an| → 0)
ทางลัดในการสอบ: สำหรับลำดับที่มีตัวประกอบ (−1)n หรือ (−1)n+1 ให้ละเครื่องหมายแล้วประเมิน |an| ถ้า |an| → 0 ลำดับลู่เข้าสู่ 0 ถ้า |an| → ค่าอื่น ต้องใช้ทฤษฎีบท 1.2.12 (ตรวจสอบพจน์คู่/คี่)

4. Finding Limits in Practice

Treat an as a continuous function f(x) and apply standard limit rules (Thm 1.2.8 above). The most important technique for rational sequences: divide numerator and denominator by the highest power of n.

Example A — Rational sequences (Limit Laws applied)

Find the limit an = (3n² + 2) / (n² + 5) Divide top and bottom by n²: = (3 + 2/n²) / (1 + 5/n²) As n → ∞, the terms 2/n² and 5/n² → 0, so: lim an = 3/1 = 3

Example B — Exponential decay

Key rule: geometric sequences limn→∞ rn = 0 if |r| < 1 limn→∞ rn = 1 if r = 1 limn→∞ rn = ∞ if r > 1 limn→∞ rn = does not exist if r ≤ −1

So an = (0.5)n → 0, and an = (−0.5)n also → 0 (since |−0.5| < 1).

Example C — Factorial and exponential growth

Useful fact n! grows faster than any exponential cn. Therefore: limn→∞ cn/n! = 0 for any constant c

4. การหาลิมิตในทางปฏิบัติ

ให้มองว่า an เป็นฟังก์ชันต่อเนื่อง f(x) แล้วใช้กฎลิมิตมาตรฐาน (ทฤษฎีบท 1.2.8) เทคนิคที่สำคัญที่สุดสำหรับลำดับเศษส่วน: หารทั้งเศษและส่วนด้วยกำลังสูงสุดของ n

ตัวอย่าง ก — ลำดับเศษส่วน (ใช้กฎของลิมิต)

หาลิมิต an = (3n² + 2) / (n² + 5) หารเศษและส่วนด้วย n²: = (3 + 2/n²) / (1 + 5/n²) เมื่อ n → ∞ พจน์ 2/n² และ 5/n² → 0 ดังนั้น: lim an = 3/1 = 3

ตัวอย่าง ข — การลดแบบเอกซ์โพเนนเชียล

กฎสำคัญ: ลำดับเรขาคณิต limn→∞ rn = 0 if |r| < 1 limn→∞ rn = 1 if r = 1 limn→∞ rn = ∞ if r > 1 limn→∞ rn = ไม่มีอยู่ if r ≤ −1

ดังนั้น an = (0.5)n → 0 และ an = (−0.5)n ก็ → 0 เช่นกัน (เนื่องจาก |−0.5| < 1)

ตัวอย่าง ค — แฟกทอเรียลและการเติบโตแบบเอกซ์โพเนนเชียล

ข้อเท็จจริงที่มีประโยชน์ n! เติบโตเร็วกว่าเอกซ์โพเนนเชียล cn ใดๆ ดังนั้น: limn→∞ cn/n! = 0 สำหรับค่าคงที่ c ใดๆ

5. The Squeezing Theorem

Sometimes an can't be evaluated directly. If you can trap it between two sequences that both converge to the same limit, it must converge there too.

Squeezing Theorem for Sequences If bn ≤ an ≤ cn for all n ≥ N and limn→∞ bn = limn→∞ cn = L then limn→∞ an = L

Example D — sin(n)/n

Find: lim sin(n) / n We know: −1 ≤ sin(n) ≤ 1 for all n Dividing by n: −1/n ≤ sin(n)/n ≤ 1/n Since lim (−1/n) = 0 and lim (1/n) = 0, by the Squeezing Theorem: lim sin(n)/n = 0

Example E — n!/nⁿ (Exam 1.2.15)

Prove: limn→∞ n!/nⁿ = 0 Write out: n!/nⁿ = (1/n)(2/n)(3/n)···(n/n) Each factor ≤ 1, and the first factor = 1/n. So: 0 ≤ n!/nⁿ ≤ 1/n Since lim (1/n) = 0 and lim 0 = 0, by the Squeezing Theorem: lim n!/nⁿ = 0
Why this matters: the n!/nⁿ example shows the Squeezing Theorem is not limited to trig — it works whenever you can find simple upper and lower bounds.

5. ทฤษฎีบทการบีบ (Squeezing Theorem)

บางครั้งหา an โดยตรงไม่ได้ แต่ถ้าสามารถบีบมันไว้ระหว่างสองลำดับที่ลู่เข้าสู่ลิมิตเดียวกัน มันก็ต้องลู่เข้าสู่ลิมิตนั้นด้วย

ทฤษฎีบทการบีบสำหรับลำดับ If bn ≤ an ≤ cn for all n ≥ N and limn→∞ bn = limn→∞ cn = L then limn→∞ an = L

ตัวอย่าง ง — sin(n)/n

หา: lim sin(n) / n ทราบว่า: −1 ≤ sin(n) ≤ 1 สำหรับ n ทุกค่า หารด้วย n: −1/n ≤ sin(n)/n ≤ 1/n เนื่องจาก lim (−1/n) = 0 และ lim (1/n) = 0, โดยทฤษฎีบทการบีบ: lim sin(n)/n = 0

ตัวอย่าง จ — n!/nⁿ (ข้อสอบ 1.2.15)

พิสูจน์: limn→∞ n!/nⁿ = 0 เขียนออกมา: n!/nⁿ = (1/n)(2/n)(3/n)···(n/n) แต่ละตัวประกอบ ≤ 1 และตัวประกอบแรก = 1/n ดังนั้น: 0 ≤ n!/nⁿ ≤ 1/n เนื่องจาก lim (1/n) = 0 และ lim 0 = 0, โดยทฤษฎีบทการบีบ: lim n!/nⁿ = 0
เหตุใดจึงสำคัญ: ตัวอย่าง n!/nⁿ แสดงว่าทฤษฎีบทการบีบไม่ได้จำกัดอยู่กับฟังก์ชันตรีโกณมิติ — ใช้ได้เสมอเมื่อหาขอบเขตบนและขอบเขตล่างง่ายๆ ได้

6. Monotone Sequences — a Shortcut for Convergence

Theorem: A sequence that is both monotone (always increasing or always decreasing) and bounded (never exceeds some fixed value M) must converge. You don't need to find L — the theorem guarantees L exists.

This is used to prove convergence without finding the exact limit. In the ค30107 quiz, you may be asked to state whether a sequence converges — this theorem is enough if the sequence is clearly monotone and bounded.

6. ลำดับโมโนโทน — ทางลัดในการพิสูจน์การลู่เข้า

ทฤษฎีบท: ลำดับที่ โมโนโทน (เพิ่มขึ้นหรือลดลงตลอด) และ มีขอบเขต (ไม่เกินค่าคงที่ M บางค่า) จะต้องลู่เข้า ไม่จำเป็นต้องหา L — ทฤษฎีบทรับประกันว่า L มีอยู่

ใช้เพื่อ พิสูจน์ การลู่เข้าโดยไม่ต้องหาลิมิตที่แน่นอน ในแบบทดสอบ ค30107 อาจถามว่าลำดับลู่เข้าหรือไม่ — ทฤษฎีบทนี้เพียงพอถ้าลำดับเป็นโมโนโทนและมีขอบเขตชัดเจน
